# Image folders

```text
categories/<category>/cover.webp
projects/<category>/<project-slug>/cover.webp
projects/<category>/<project-slug>/gallery-01.webp
projects/<category>/<project-slug>/gallery-02.webp
...
```

Recommended export: WebP, sRGB, 1600–2000 px on the longest edge, generally under 400 KB per image.

Keep folder names and file names lowercase, use hyphens instead of spaces, and update `data/categories.js` or `data/projects.js` after adding images.
